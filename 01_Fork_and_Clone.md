# Part 1: Create from Template

**Time: 1 minute**

> **New here?** Read [The Manifesto](./00_Manifesto.md) first.

## Step 1: Create Repository from Template

1. Go to [https://github.com/agrana/free_personal_stack](https://github.com/agrana/free_personal_stack)
2. Click **"Use this template"** button (green button near the top right)
3. Select **"Create a new repository"**
4. Fill in:
   - **Repository name**: Choose a name for your project
   - **Description**: Optional
   - **Visibility**: Public or Private (your choice)
5. Click **"Create repository from template"**

That's it. You don't need to clone anything locally. Everything runs from GitHub.
